🌷 Bloom & Do
Application simple et élégante de gestion de tâches (to-do list), pensée pour rester focus sans distraction.

🚀 Fonctionnalités
➕ Ajouter une tâche
✅ Marquer une tâche comme terminée
❌ Supprimer une tâche
💾 Sauvegarde automatique avec localStorage
♿ Accessibilité (ARIA, messages dynamiques)
🌸 Interface simple et agréable

🛠️ Technologies utilisées
HTML5
CSS3
JavaScript (Vanilla)
LocalStorage (navigateur)

📂 Structure du projet
bloom-do/
│── index.html
│── style.css
│── script.js

💡 Objectif du projet
Ce projet a été réalisé pour :
pratiquer JavaScript (DOM, events, localStorage)
créer une interface accessible
développer une mini app utile et propre
